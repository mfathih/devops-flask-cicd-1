1) Command ran :
curl http://localhost:5050/health

Error message :
curl: (56) Recv failure: Connection reset by peer

Reason : Your Flask app runs inside the container on port 5000, but your compose is mapping: "5050:5050"
So Docker is sending traffic to container port 5050, where nothing is listening.

Fix : Edit docker-compose.yaml
    ports:
      - "5050:5000"

Verify :
curl http://localhost:5050/health

Output should be {"status":"healthy"}
Go to http://127.0.0.1:5050 on browser

Notes : On the Flask container description, it says "Running on http://127.0.0.1:5000"
